# List of Example Scripts

#### `add_dipole.py`
Add a dipole of given magnitude and direction to an SKA map using ModulatedMapGenerator and fit.

#### `fixed_dipole.py`
Fit a dipole with fixed coordinates to an SKA map with using a custom prior.

#### `masking_downgrading.py`
Change the resolution of an SKA map and mask the equatorial poles and Galactic plane.
